# Cheating Detection Website

## Setup Guide

### 1. Install MongoDB

- Download and install MongoDB
- Open MongoDB and make sure the **local server is running**

---

### 2. Start Backend

- Open the **backend folder**
- Run this command:

```bash
node --inspect server.js
```

---

### 3. Start Frontend

- Go to the **main project folder**
- Right-click → open terminal
- Run:

```bash
npx vite
```
